# 项目概述 java796

本项目为一款基于SpringBoot+Vue+uniapp技术的二手商城系统，适用于校园环境，提供二手商品的交易功能。

## 技术栈

- **开发语言**：Java
- **开发框架**：SpringBoot
- **前端框架**：Vue、ElementUI
- **跨平台应用**：uniapp、小程序
- **数据库**：MySQL
- **开发工具**：IDEA、JDK1.8、Maven、Node.js

## 功能模块

### 用户模块

- 用户登录
- 用户注册
- 忘记密码

### 用户前台

- 首页轮播图展示
- 公告信息浏览
- 系统简介查看
- 二手商品浏览与搜索
- 我的交易记录
- 商品评价
- 发布二手商品
- 个人中心
- 修改密码
- 发布帖子
- 我的发帖
- 校园论坛交流

### 后台管理

- 首页数据统计
- 个人中心管理
- 用户管理
- 商品分类管理
- 二手物品管理
- 物品评价管理
- 交易管理
- 校园论坛管理
- 轮播图管理
- 公告管理
- 系统简介管理

## 运行环境

- 服务器：推荐使用Linux系统
- 数据库：MySQL（版本5.7+）
- IDE：IntelliJ IDEA、WebStorm、Visual Studio Code等支持Java和JavaScript的IDE

## 部署步骤

（具体部署步骤将根据具体环境配置详细说明）

1. 环境准备：确保JDK、Maven、Node.js等环境安装正确
2. 数据库搭建：安装MySQL，导入初始数据
3. 后端部署：编译后端代码，部署至服务器
4. 前端部署：编译前端代码，部署至服务器
5. 小程序/uniapp部署：根据开发平台要求部署至相应平台
